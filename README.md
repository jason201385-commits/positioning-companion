# 🧭 定位小書僮

> **在 AI 時代，先判斷「需不需要換方向」，再決定做什麼。**
>
> 這是一個開源自助定位工具。它用訪談與反向 gate，幫你判斷比較適合做內容、改善工作流、留在本業放大優勢，或暫不定位。

完整方法裡，「定位」負責想清楚與設計驗證；「陪跑」負責後續執行與問責。**本開源包完整提供定位，不要求購買陪跑，也不把做內容、轉行或做產品當成標準答案。**

---

## 跑完會拿到什麼

你會拿到一張 `positioning-card/v2` 定位卡，至少包含：

- `mode`：`content`、`workflow`、`business-amplifier` 或 `pause`。
- `gate_status`：`pass`、`partial` 或 `fail`，以及沒通過的理由。
- `[使用者原話]` 與 `[AI 建議]`：清楚分開，不讓 AI 的推論冒充你說過的話。
- `no_product_alternative`：不用做新產品、甚至不用本產品也能前進的方法。
- `first_evidence`：第一個可觀察、可驗收的里程碑。
- `falsifiable_check`：哪個結果出現就要改寫、回復或停止。
- `next_step`：先停止、做兩週小實驗，或採取一個可逆的小行動。

有效結論可以是：

- 做內容，但先驗證受眾與問題。
- 不做內容，先解一個工作流。
- 不轉行，用 AI 放大本業。
- 不做產品，先用人工服務、既有工具或範本解題。
- 現在先停止或暫不定位。

## Quick 與 Full

- **Quick 快速草稿**：先從少量但具體的證據形成方向假設。證據不夠時會標 `gate_status: partial`，不假裝已驗證。
- **Full 完整驗證**：補齊七面向訪談、反向 gate、受眾或工作流證據，以及最小實驗。

兩種模式都不宣稱固定完成時間；你可以先 quick，再視需要補成 full。

## 三種使用方式

### A. 貼進任何 AI（不用安裝）

1. 打開 [`一鍵定位-貼上任何AI.md`](一鍵定位-貼上任何AI.md)。
2. 整份複製到 ChatGPT、Claude、Gemini 或其他可長對話的 AI。
3. 輸入「開始」，再選 quick 或 full。

一鍵版已包含完整的反向 gate、方向庫、定位卡 schema 與純文字 fallback，不依賴其他工具。

### B. 安裝成 Claude Code Skill

最省事的方式：把 repo 網址貼給 Claude Code，說「幫我安裝這個 skill」。

手動安裝時，把 repo 內容放進：

```text
~/.claude/skills/positioning-companion/
```

確認最終檔案位置是：

```text
~/.claude/skills/positioning-companion/SKILL.md
```

Windows 範例：

```text
C:\Users\<你>\.claude\skills\positioning-companion\SKILL.md
```

重新開啟 Claude Code 後，說「幫我定位」「我在 AI 時代能做什麼」或「我需要做內容或產品嗎」。

## 反向 gate 怎麼保護你

工具在推薦方向前會先問：

- 你真的需要換方向嗎？
- 你真的需要做內容嗎？
- 你真的需要做新產品嗎？
- 財務、資料安全與時間條件允許嗎？
- 哪個真實結果會推翻這個建議？

外部內容或服務方向會使用「10 個真名」作為操作測試：能否列出真實、有名有姓的可能受眾。工作流型或本業放大型若不需要外部受眾，可標 `not_applicable` 並寫理由，不必硬湊名字。

## 不裝任何配套也能完成

本包不要求 `social-post`、`find-community`、`content-flywheel` 或其他不存在於 repo 的工具。

純文字 fallback：

- **找受眾**：直接列 10 個真名，記錄每個人是否真的有這個痛、目前怎麼解。
- **寫第一份內容**：寫四行——對誰說、他卡在哪、你的證據、一個可做動作。
- **驗證工作流**：用紙筆或試算表記錄原流程、測試流程、例外與回復結果。
- **驗證產品**：先人工交付一次，觀察需求是否重複，不先寫完整產品。

## FAQ

**這跟性格測驗有什麼不同？**

測驗通常回傳分類；定位小書僮會要求具體場景、分開本人原話與 AI 建議，並設計能推翻結論的真實檢查。方向可以被修改，也可以被拒絕。

**它會幫我賺錢嗎？**

不保證。它只幫你形成較清楚、可驗證的判斷。市場是否買單，要看真實受眾與交付證據。

**經濟壓力大可以用嗎？**

可以用來釐清，但工具不會替你做辭職、借錢或投資決定。若新方向會危及基本現金流，合法輸出是 `gate_status: fail` 或 `mode: pause`，先穩住現況。

**一定要公開定位卡嗎？**

不用。公開前應移除公司、主管、同事、客戶、財務與其他可識別資訊；也可以只保留給自己。
