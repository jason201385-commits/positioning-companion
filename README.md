# 定位小書僮（開源自助版）

> **定位為主、陪跑為輔**——這是上游的「定位」工具（先幫你想清楚方向）；定位完成後的「陪跑」是下游、可選的步驟（見文末）。

**一句話**：一個幫你「在 AI 時代釐清自己該做什麼」的定位工具——用你的好惡、夢想年、財務可行性，問清楚你「想逃離什麼」，再選一條你撐得住、又有機會變現的賽道。

**給誰用**：
- 想用 AI 開始做點什麼，但不知道從哪裡開始的人。
- 一直在想方向、卻遲遲沒收斂出「我到底該做哪一條」的人。
- 手上有專業 / 經歷 / 興趣很多元，反而不知道該押哪一個的人。

> 這不是教你「學哪個 AI 工具」，而是先幫你想清楚「你該往哪走」。方向想清楚了，後面的努力才不會白費。

---

## 這是什麼

`定位小書僮` 是一個 **Claude Code skill**。它把「釐清一個人 → 給他相對合適的方向」這套訪談式定位流程，寫成 Claude 可以照著跑的提示與步驟。

核心產出是三樣東西：

1. **核心痛點一句話**——你真正想擺脫的是什麼（不是你以為你想要的）。
2. **身分宣言**——一句「你是一個 ___ 的 ___」，當作你後續方向的定錨。
3. **選定賽道（2-3 條候選）**——讓你自己挑，工具給判斷、不替你拍板。

> 跑完還會做兩道把關（見 `SKILL.md` 完成定義）：講得出 10 個真實受眾、第 91 天接得上變現鉤——確保方向不只是「想清楚」，而是「接得上現實」。

## 怎麼用（self-serve）

### 方式 A：當成 Claude Code skill（推薦）

1. 把整個 repo 內容複製成一個叫 `定位小書僮` 的資料夾（內含 `SKILL.md`、`references/`、`runners/`），放到 `~/.claude/skills/` 下——最終路徑要長成 `~/.claude/skills/定位小書僮/SKILL.md`（Windows：`C:\Users\<你>\.claude\skills\定位小書僮\SKILL.md`）。資料夾名請用 `定位小書僮`，跟 skill 名一致。`runners/` 是工作目錄，跑的時候對象存檔會寫在這裡。
2. 開 Claude Code，直接說：**「幫我定位」** 或 **「我在 AI 時代能做什麼」**。
3. 跟著它的提問走就好。

### 方式 B：把 references 當提示直接用

不想裝 skill 也可以。打開 `references/` 裡的檔，把內容貼給任何一個夠強的 AI 對話視窗，請它「照這份流程帶我跑一遍定位」即可。

### 一步步流程

1. **跑定位引擎** → `references/定位引擎.md`
   主幹方法：用好惡先行、低門檻地開場，帶出你想逃離什麼。

2. **回答好惡 / 夢想年 / 財務** → `references/01-intake.md`
   - **好惡**：你喜歡什麼、特別受不了什麼（厭惡是硬約束，受不了的事不勉強）。
   - **夢想年題**：「有一年、夠多的錢和時間，但不能工作只能花錢，你會怎麼規劃？」——你想做的＝興趣方向，你刻意避開的＝間接痛點。
   - **財務可行性**：先看現金流壓力。壓力大，先穩住現金流再衝；這是給你判斷框架，不是替你決定辭職或借錢。

3. **選賽道** → `references/02-choose-lane.md` ＋ `references/賽道庫.md`
   用「興趣 × 技能 × 厭惡 × 現金流」交叉，先砍掉你做不到的，再給你 2-3 條相對合適的候選讓你自己挑。`賽道庫.md` 內建 8 個 AI 創業職業（內容、自動化、教學、視覺、產品、行銷、外包、顧問）當現成候選，每個都附一個起步參考。

4. **得到你的結果**：核心痛點一句話 ＋ 身分宣言 ＋ 選定賽道。

> 跑完這一輪，你手上會有一個「方向」，而不是一堆「待考慮的可能」。

## 它承諾什麼 / 不承諾什麼

**會幫你**：
- 把方向想得更清楚——你到底想逃離什麼、最該走哪一條。
- 判斷某個方向**適不適合你**（你的好惡、現金流、撐不撐得住）。
- 提醒你某個方向**市場買不買單**（有沒有人有同樣的痛、你有沒有降維空間）。

**不會幫你**：
- **不保證變現**、不保證做了就接得到案、賺得到錢。
- 不替你做財務決定（辭職 / 借錢 / 投資），只給你判斷框架。
- 不給你「跟著做 N 天就保證 X」這種承諾。

> 與其自己一個人想一年，不如花一個下午用它先看清楚。看清楚之後要不要做、怎麼做，決定權都在你。

定位只是起點。如果你跑完之後想把這個方向變成一份**具體的 90 天行動計畫**、而且想有人陪你撐過去、卡住時有人問責——

可以找 **Jason（魯班）/ easyknowai** 聊聊「陪跑」。這套開源工具是上游的「定位」；把定位變成每天做得到、撐得住、最後串得到現金流的行動，是陪跑在做的事。

> 開源版本就是完整的定位工具，不裝任何功能在後面。要不要進一步找人陪跑，完全看你自己。
