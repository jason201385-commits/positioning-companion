# 🧭 定位小書僮

> **在 AI 時代，先想清楚「該做什麼」，再開始做。**
> 一個自助定位工具：用你的好惡、夢想年、財務可行性，幫你選一條撐得住、又有機會變現的方向。（定位為主、陪跑為輔）
>
> ℹ️ 名稱說明：完整方法叫「陪跑」＝定位（想清楚方向）＋陪跑（每天執行、有人問責）。**本開源包只做「定位」這一段**；下游的「陪跑」是定位完成後的可選步驟，不在本包。

把抽象的「我到底該幹嘛」收斂成三樣具體的東西。一個下午，自己跑，不用先學任何 AI 工具。

## 👤 給誰用

| 你是這樣的人 | 你卡在哪 |
|---|---|
| 想用 AI 做點什麼，但不知道從哪開始 | 工具一堆，方向是空的 |
| 一直在想方向，卻遲遲收斂不出「該做哪一條」 | 想了半年，還是一堆「待考慮」 |
| 專業／經歷／興趣很多元，反而不知道押哪個 | 樣樣會一點，不知道哪個能變現 |
| 想做內容或個人品牌，但「不知道該寫什麼」 | 想開始，卻不知道從哪個身分開口 |

> 這不是教你「學哪個 AI 工具」，而是先幫你想清楚「你該往哪走」。方向對了，後面的努力才不會白費。

## 🎁 跑完你會拿到什麼（三件套）

1. **核心痛點一句話** —— 你真正想擺脫的是什麼（不是你以為你想要的）。
2. **身分宣言** —— 一句「你是一個 ___ 的 ___」，當作你後續方向的定錨。
3. **2-3 條選定賽道** —— 工具給判斷，讓你自己挑，不替你拍板。

> 還有兩道把關，確保方向接得上現實，不只是「想清楚」：
> **① 10 個真名**——講得出 10 個真實、會為這條方向停下來追蹤／付費的人；
> **② 變現鉤**——這條方向接得上你「現有的賺錢能力」。過不了 → 回去重挖。

## ⭐ 三種用法（從最低門檻開始）

### 🟢 用法 A：貼進任何 AI 對話視窗（最簡單，不用裝任何東西）

> **沒有 Claude Code 也能跑。** ChatGPT、Claude.ai、Gemini 都行。

1. 打開 `一鍵定位-貼上任何AI.md`，**整份複製**。
2. 貼進任何一個 AI 對話視窗。
3. 對它說一句：**「開始」**。
4. 跟著它的提問走就好。

### 🔵 用法 B：當 Claude Code skill

1. 把整個 repo 複製成一個叫 `定位小書僮` 的資料夾，放到 `~/.claude/skills/` 下
   （最終路徑長成 `~/.claude/skills/定位小書僮/SKILL.md`；Windows：`C:\Users\<你>\.claude\skills\定位小書僮\SKILL.md`）。
2. 開 Claude Code，直接說：**「幫我定位」** 或 **「我在 AI 時代能做什麼」**。
3. 跟著它的提問走。`runners/` 是工作目錄，你的存檔會寫在這裡。

### 🟣 用法 C：自己讀 references/，貼給 AI 帶你跑

不想用上面兩種也行。打開 `references/` 裡的四個檔，貼給任何一個夠強的 AI，請它「照這份流程帶我跑一遍定位」：

| 檔 | 做什麼 |
|---|---|
| `references/定位引擎.md` | 定位主幹（好惡 + 夢想年 + 財務可行性，最短路徑） |
| `references/01-intake.md` | 七面向逐項問法 + 收斂格式（想更深就用這個） |
| `references/02-choose-lane.md` | 選賽道引擎（四維砍 → 三篩 → 把關 + 10 真名） |
| `references/賽道庫.md` | 8 個 AI 創業職業現成候選 |

## 💬 跑起來長怎樣

```
教練：先不談你想學什麼。如果有一年，有夠多的錢和時間，
      但「不能工作、只能花錢」，你會想怎麼規劃這一年？
你　：大概到處走走、看很多書，可能會自己拍點東西記錄下來。
教練：「拍點東西記錄下來」——上一次你這樣做是什麼情況？給我一個畫面。
你　：之前幫朋友的小店拍了短影片，他客人變多，我那陣子睡得很好。
教練：注意到了嗎？你刻意避開「替別人打工」，但一講到「幫小店被看見」就亮了。
      這條線，我們往下挖一輪——
```

> 它不會一題就下結論。一個答案是【線索】，不是【結論】；至少跨 2-3 輪，把好惡、夢想年、技能、現金流兜在一起，才收斂。

## ❓ FAQ

**這跟線上測驗 / 性格測驗有什麼不同？**
測驗給你一個標籤就結束了。這個工具是一場**訪談**：它會追問、要你給具體畫面、把你的厭惡當硬約束砍掉撐不住的選項，最後收斂成「方向 + 為什麼是這條」，而不是一個分類結果。

**要付費嗎？**
不用。MIT 授權，開源自助版完整免費，歡迎自用、改造、轉給有需要的人。

**沒有 Claude Code 能用嗎？**
能。用「**用法 A**」——把 `一鍵定位-貼上任何AI.md` 整份貼進 ChatGPT／Claude.ai／Gemini，說「開始」即可。Claude Code 只是其中一種跑法，不是必要條件。

**它會幫我賺錢嗎？**
**不保證。** 它幫你把方向想清楚，並提醒你某條路「市場買不買單」；但要不要做、做不做得成、賺不賺得到，決定權和執行都在你。內容／接案的回報是複利，不是急救——這點先講清楚。

**我現在經濟壓力很大，適合用嗎？**
可以用來**想清楚方向**，但工具會明確建議你：壓力大、沒存款時，先把現金流穩住（找份工作或打工都行，給自己「大概半年沒收入也餓不死」的底）再認真投入。它**不會**替你決定辭職／借錢／投資——那是你自己的決定，必要時請諮詢專業。

## ✅ 會幫你 / ❌ 不會幫你

**✅ 會幫你**
- 把方向想清楚——你到底想逃離什麼、最該走哪一條。
- 判斷某個方向**適不適合你**（你的好惡、現金流、撐不撐得住）。
- 提醒你某個方向**市場買不買單**（有沒有人有同樣的痛、你有沒有降維空間）。

**❌ 不會幫你**
- **不保證變現**，不保證做了就接得到案、賺得到錢。
- 不替你做財務決定（辭職／借錢／投資），只給你判斷框架。
- 不給你「跟著做 N 天就保證 X」這種承諾。

> 與其自己一個人想一年，不如花一個下午先看清楚。看清楚之後要不要做、怎麼做，決定權都在你。
