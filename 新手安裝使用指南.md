# 🚀 新手安裝使用指南 — 從零開始，一步一步來

> 這份是給**完全新手**的。看不懂電腦沒關係，我們一步一步走。卡住是正常的，大家都是這樣過來的。

---

## 先搞清楚：你需要裝東西嗎？

兩條路，看你要哪一種——**先從路 A 試，覺得不夠再走路 B。**

---

## 🟢 路 A · 完全不用裝（5 分鐘，推薦先試這個）

只想**先把方向想清楚**？連軟體都不用裝：

1. 打開你**已經在用的任何 AI**——ChatGPT、[Claude.ai](https://claude.ai)、Gemini 都行。
2. 到 [`一鍵定位-貼上任何AI.md`](一鍵定位-貼上任何AI.md)（或示範頁上的「一鍵複製」按鈕），**整份複製**。
3. 貼進 AI 對話框，送出。
4. 它會像朋友一樣一題一題問你。聊十幾分鐘，你就拿到一張「定位卡」。

✅ **適合**：只想搞清楚「我在 AI 時代該做什麼」的人。零門檻、零安裝。

---

## 🔵 路 B · 裝 Claude Code（完整功能：定位 + 每天自動發文/生圖/剪片）

定位完想**真的每天執行**——自動發文、生圖、剪影片——就需要裝 Claude Code，把定位小書僮和配套工具都變成隨叫隨用的「skill」。

### Step 1｜下載、安裝 Claude Code
👉 **照這份逐步教學（有圖、一步一步）：<https://claude.easyknowai.com/>**

跟著做就好，不用先懂原理。裝好你會有一個可以打字跟它對話、還能幫你操作電腦的 Claude。

### Step 2｜選方案（先用最省的）
- **新手先訂 Pro 就好**，**不用**另外買 API。
- 之後你認真在跑、額度不夠用了，再升級 Max。
- 額度用完別緊張——**有用完都是好事**，代表你真的有在做。

### Step 3｜安裝 skill（「給我裝」三字訣）
把想裝的工具**網址貼給 Claude**，跟它說一句：**「幫我安裝這個」**。它看不懂你就回：**「看不懂，幫我裝」**。

先裝**定位小書僮**（本工具）：
```
https://github.com/jason201385-commits/positioning-companion
幫我安裝這個 skill
```

定位完想每天執行，再裝這幾個配套工具（[Hao 前輩](https://github.com/Hao0321)開源，MIT）：

| 工具 | 幹嘛用 |
|---|---|
| [social-post](https://github.com/Hao0321/claude-skill-social-post) | 學你的語氣、排 14 天內容、自動發到 FB／IG／Threads／X |
| [ai-media-generator](https://github.com/Hao0321/ai-media-generator) | 生圖、生影片、生音樂的專業提示詞 |
| [video-autopilot-kit](https://github.com/Hao0321/video-autopilot-kit) | 自動剪短影音 |
| [code-cleanup](https://github.com/Hao0321/claude-skill-code-cleanup) | 幫你把 skill／專案保持乾淨 |

### Step 4｜卡住怎麼辦（一定會卡，正常的）
- **每一步不確定，就「截圖」問 AI**——把畫面貼給 Claude、或另開一個 ChatGPT 問「這步我該做什麼」，它會一步步帶你。
- 真的卡住 → 貼到**自由工坊 Discord**（下面），有人幫你看。
- **學就完事**，不用一次全懂。

---

## 🎯 裝完，你的一天長怎樣

```
定位小書僮  →  想清楚今天/這季要走的方向
     ↓ 交棒
配套工具    →  把方向變成每天的產出（發文 / 生圖 / 剪片）
```

定位是**輔助**：它讓你不會「有一堆神兵、卻不知道要做什麼」而空轉。

---

## 🆘 新手三個常見卡點（先講在前頭，遇到別慌）

| 遇到 | 怎麼辦 |
|---|---|
| 畫面全是英文、看不懂 | 截圖問 AI「這一步我該做什麼」，它會用中文教你 |
| 額度好像用完了 | 有用完都是好事＝你有在做；先 Pro 撐，需要再 Max |
| 裝到一半報錯（git、資料夾…） | 截圖問 AI 就好，它會幫你補齊。大家都卡過這關 |
